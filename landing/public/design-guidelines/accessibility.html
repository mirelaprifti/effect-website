Color Contrast

WCAG AA Standards

Normal Text 4.5:1

Minimum contrast ratio for body text and UI components

Large Text 3:1

Minimum for text 18pt+ regular or 14pt+ bold

UI Components 3:1

Minimum for active UI components and graphics
