Spacing Principles

• Always use multiples of 8px (or 4px for fine adjustments)
• Maintain consistent vertical rhythm
• Scale spacing with content importance
• Consider responsive adaptability

Spacing & Corner Best Practices

Spacing Guidelines

Use consistent 8px increments for all spacing
Reduce padding by 8px for each nesting level
Maintain visual hierarchy through spacing
Consider touch targets on mobile (min 44px)

Corner Radius Guidelines

Use 8px (radius-md) as the default for most elements
Reduce radius by 4px for each nesting level
Use radius-full for pills and circular elements
Maintain consistency within component families
