Accessibility

• WCAG AA compliant contrast ratios
• Color-blind friendly combinations
• Semantic meaning beyond color
• High contrast for readability

Usage Principles

• Neutral tones for primary interface
• Semantic colors for status feedback
• Consistent dark mode adaptation
• Minimal color for maximum impact

Implementation

• Tailwind CSS color classes
• CSS custom properties
• Design token system
• Automatic dark mode switching
