Getting Started with Effect

Learn the fundamentals of Effect and build your first application

Section Header

Installation

Install Effect using your preferred package manager

Body Text

Effect is a powerful library for building robust TypeScript applications. It provides a comprehensive toolkit for handling errors, managing dependencies, and composing complex asynchronous operations in a type-safe manner.

With Effect, you can write more maintainable code by leveraging functional programming patterns while maintaining the familiarity and ecosystem of TypeScript.

Code Block

import { Effect } from "effect"

const program = Effect.gen(function* () {
  const name = yield* Effect.succeed("Effect")
  const message = `Hello, ${name}!`
  return message
})

Effect.runSync(program) // "Hello, Effect!"

Guidelines

Best Practices

  • Use the type scale consistently across all pages
  • Maintain clear hierarchy with appropriate font weights
  • Limit the number of font sizes on a single page
  • Use Space Mono exclusively for code and technical content
  • Ensure sufficient contrast between text and background
  • Keep line lengths between 50-75 characters for optimal readability

Accessibility

  • Minimum font size of 14px (0.875rem) for body text
  • WCAG AA contrast ratios: 4.5:1 for normal text, 3:1 for large text
  • Use semantic HTML elements (h1-h6) for proper structure
  • Avoid using color alone to convey meaning
